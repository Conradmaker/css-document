# CSS 기본

CSS는 HTML의 기본문법을 어느정도 숙지하고 공부하시기를 추천드려요!

선택자는 다음과 같이 사용할 수 있습니다.

```css
선택자 {
 속성: 속성값;
 속성: 속성값;
}
```

​

### **선택자\(selector\)의 역할**

**선택자의 역할은 스타일을 적용하고 싶은 영역을 선택해주는 역할을 합니다.**

```css
div{                 //div 영역을 선택
  color: red;
}
```

### \*\*\*\*

### **속성\(Property\)과 값\(Value\)의 역할**

**'속성'**은 스타일의 무엇을 바꿀것인지, 

**'값'**은 어떻게 바꿀것인지를 정의해 줍니다.

 `color: 속성    red:값`

```css
div{
    color: red;
    font-size: 20px;
    font-weight: bold;
}
/* 글자색:빨강;
글자크기:20px;
글자두께:두껍게; */
```
